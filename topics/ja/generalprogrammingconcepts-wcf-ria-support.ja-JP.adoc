= WCF RIA のサポート

このセクションは、{ProductName} で Microsoft RIA サービスのサポートの概要を提供します。

Microsoft RIA サービスでは、DomainDataSource と呼ばれるデータ ソース コントロールを使用してアプリケーションのクライアントおよびサーバーのデータ転送を作成することができます。このコントロールを使用すると、開発者が簡単に RIA サービスによって生成されたミドルウェアと通信することができます。また、サーバー側データをクエリすることも可能です。さらに、データ応答をページングするメカニズムを提供します。これによって、データ転送のオーバーヘッドおよびデータのグループ化およびフィルタリングを低くします。

複数 Infragistics コントロールは WCF RIA サービスと互換性があります。
