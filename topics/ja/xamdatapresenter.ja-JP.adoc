= xamDataPresenter

== このグループのトピックについて

=== 概要

このセクションには、WPF の  _xamDataPresenter_™ コントロールについてのトピックが含まれています。

_xamDataPresenter_   はレイアウト コントロールです。データを複数のビューおよびカスタム ビューでの視覚化を許可します。動的にデータ ビューを変更する場合に便利です。

以下の画像は定義済みのビューを表示します。注: 画像に複数のビューがありますが、コントロールは一度にビューを 1 つのみ表示します。
