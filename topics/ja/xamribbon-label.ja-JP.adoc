= ラベル





シンプルなテキスト ラベルが必要である場合、Label ツールが適切なツールです。Label ツールはユーザー操作をサポートしません。事実、Label ツールの機能は表示目的のみなので、KeyTip プロパティを設定しても Label ツールはキーチップを表示しません。

Label ツールは IRibbonTool インターフェイスも実装します。これは、高度な xamRibbon 機能を容易にするためのクローンおよびバインド機能を追加します。ユーザー独自のカスタム xamRibbon ツールを作成していなければ、IRibbonTool インターフェイスについての心配は不要です。

link:{ApiPlatform}ribbon{ApiVersion}~infragistics.windows.ribbon.labeltool.html[LabelTool] クラスは、xamRibbon で容易にラベルを管理する上で役立つ複数の追加プロパティを公開します。LabelTool は、プロパティが設定された場合、またはラベルが特定の状態にある場合に確認を手助けするために複数の 'Has' および 'Is' プロパティを公開します。xamRibbon で Label ツールを設定する際に役立つ標準プロパティには以下が含まれます。

* link:{ApiPlatform}ribbon{ApiVersion}~infragistics.windows.ribbon.labeltool~caption.html[Caption] - このテキストは、解決された SizingMode プロパティに関係なく Label ツールの隣りに常に表示されます。
* link:{ApiPlatform}ribbon{ApiVersion}~infragistics.windows.ribbon.labeltool~id.html[Id] - Id は不要ですが、ランタイムに link:xamribbon-access-a-tool-by-its-id.html[ツールにアクセス]する手助けとなります。
* link:{ApiPlatform}ribbon{ApiVersion}~infragistics.windows.ribbon.labeltool~largeimage.html[LargeImage] - 通常は 32 x 32 ピクセルの画像です。ツールの SizingMode が ImageAndTextLarge に設定される場合にこの画像が表示されます。
* link:{ApiPlatform}ribbon{ApiVersion}~infragistics.windows.ribbon.labeltool~sizingmode.html[SizingMode] - このプロパティは読み取り専用で、ツールが関連付けられたリボン グループで現在どのようにサイズ設定されているかを返します。link:xamribbon-resize-a-tool.html[ツールのサイズを設定]したい場合、RibbonGroup の link:{ApiPlatform}ribbon{ApiVersion}~infragistics.windows.ribbon.ribbongroup~minimumsizeproperty.html[MinimumSize] および/または link:{ApiPlatform}ribbon{ApiVersion}~infragistics.windows.ribbon.ribbongroup~maximumsizeproperty.html[MaximumSize] 添付プロパティを設定できます。
* link:{ApiPlatform}ribbon{ApiVersion}~infragistics.windows.ribbon.labeltool~smallimage.html[SmallImage] - 通常は 16 x 16 ピクセルの画像です。ツールの SizingMode が ImageAndTextNormal に設定される場合にこの画像が表示されます。
