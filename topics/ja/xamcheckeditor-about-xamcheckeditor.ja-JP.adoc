= xamCheckEditor について

xamCheckEditor™ コントロールは、True、False、および Indeterminate の各状態をサポートするシンプルなチェックボックス コントロールです。XamCheckEditor は、xamDataGrid™ コントロールの Field のように別のコントロールの中に組み込まれて通常使用されます。ただし、スタンドアロン コントロールとしても使用できます。xamDataGrid の Field のように、ブール値で複数のレコードに素早くマークするためのより簡単な方法をエンドユーザーに提供することができます。
