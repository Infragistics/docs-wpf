= 右から左の双方向言語のサポート

Right-to-Left (RTL) および Bi-Directional (BiDi) 言語のサポートで、{PlatformName} アプリケーションをよりグローバルにすることができます。ローカライズ機能の利点を活用するために、xamGrid オブジェクトの FlowDirection プロパティを以下の FlowDirection 列挙体のいずれかの値に設定します。

* LeftToRight
* RightToLeft

*XAML の場合:*

----
<ig:XamGrid x:Name="MyDataGrid" FlowDirection="RightToLeft" ItemsSource={StaticResource DataUtil} 
      AutoGenerateColumns="True">
      …
</ig:XamGrid>
----

*Visual Basic の場合:*

----
Me.MyDataGrid.FlowDirection = Windows.FlowDirection.RightToLeft
----

*C# の場合:*

----
this.MyDataGrid.FlowDirection = System.Windows.FlowDirection.RightToLeft;
----
