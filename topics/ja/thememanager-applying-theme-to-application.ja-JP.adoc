== アプリケーションへのテーマの適用

[[_Hlk368069110]]

=== 概要

link:{RootAssembly}{ApiVersion}~infragistics.themes.thememanager.html[ThemeManager] link:{RootAssembly}{ApiVersion}~infragistics.themes.thememanager~applicationtheme.html[ApplicationTheme] プロパティを使用して、アプリケーションにテーマを適用します。

テーマは、サポートされている共通の Microsoft WPF コントロールなど、アプリケーションのすべてのコントロールに適用されます。

注:

[NOTE]
====
`ApplicationTheme` プロパティは、コード ビハインドでのみ設定されます。
====

=== プロパティ設定

以下の表は、任意の構成とそれを管理するプロパティ設定のマッピングを示します。

[options="header", cols="a,a,a"]
|====
|目的:|使用するプロパティ:|設定値:

|WPF アプアリケーション全体へのテーマの適用
| link:{RootAssembly}{ApiVersion}~infragistics.themes.thememanager~applicationtheme.html[ApplicationTheme]
| link:{RootAssembly}{ApiVersion}~infragistics.themes.themebase_members.html[ThemeBase]

|====

[[_Hlk337817761]]

=== 例

以下のコードは、 _Office2013_    __のテーマ__ をアプリケーションに適用する方法を示します。

*C# の場合:*

[source,csharp]
----
ThemeManager.ApplicationTheme = new Office2013Theme();
----

*Visual Basic の場合:*

[source,vb]
----
ThemeManager.ApplicationTheme = New Office2013Theme()
----
