= デザイナー ガイド

デザイナー ガイドは、スタイルとデザイナーのリソースです。このガイドは、{ProductName} コントロールのさまざまなスタイリングの要点を学習する支援をします。アプリケーションの見栄えをよくするためにさまざまな技術を使用することによってデザイナーの教育に役立つトピックだけでなく、各コントロールのさまざまなターゲットタイプを例示する略図があります。このガイドに含まれる情報を使用すると、{ProductName} で強化されたアプリケーションで優れたスタイリングを達成することが可能となります。
