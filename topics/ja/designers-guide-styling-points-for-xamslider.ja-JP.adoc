= xamSlider のスタイリング ポイント

== xamSlider プロパティ

以下の図は、xamSlider によって使用されるさまざまなコントロールを取り上げます。特定の項目を再スタイルするには、最初に図で TargetType を識別し、次にその後の表で対応する Style プロパティを識別します。

image::images/DesignersGuide_Styling_Points_for_xamSlider_01.png[]

== xamSlider スタイル プロパティおよび TargetType

以下の表は xamSlider コントロールによって使用されるさまざまなセル コントロールをリストし、カスタム スタイルを設定するために使用できるプロパティを識別します。

[options="header", cols="a,a,a"]
|====
|TargetType|Style プロパティ|説明

|xamSlider
|xamSlider.Style
|xamSlider コントロールをスタイルします。

|TrackFill
|xamSlider.TrackFillStyle
|xamSlider のすべての TrackFill をスタイルします。

|
|Thumb.TrackfillStyle
|特定の Thumb の TrackFill をスタイルします。

|xamSliderThumbBase
|xamSlider.ThumbStyle
|xamSlider のすべての Thumb をスタイルします。

|
|xamSliderThumbBase.Style
|Thumb をスタイルします。

|====
