= 並べ替え

並べ替えは xamGrid が提供する重要な機能です。xamGrid コントロールは、ユーザーに代わって並べ替えを自動的に表示および処理します。エンド ユーザーは、列ヘッダーをクリックすることによって列を並べ替えでき、これによって希望の順序でグリッド データを表示できます。

デフォルトで、xamGrid において並べ替えは有効です。ただし、以下のコードが示すように、xamGrid で並べ替えを無効にすることができます。

*XAML の場合:*

----
<ig:XamGrid.SortingSettings>
   <ig:SortingSettings AllowSorting="/>
</ig:XamGrid.SortingSettings>
----

*Visual Basic の場合:*

----
Me.MyGrid.SortingSettings.AllowSorting = False
----

*C# の場合:*

----
this.MyGrid.SortingSettings.AllowSorting = false;
----
