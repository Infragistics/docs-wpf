= 行セレクター イベント

xamGrid には、行セレクターに関連付けられたイベントが 1 つあります。このイベントによって、行セレクターがクリックされた後でいくつかのカスタム ビジネス ロジックを実行することができます。

link:{ApiPlatform}controls.grids.xamgrid{ApiVersion}~infragistics.controls.grids.xamgrid~rowselectorclicked_ev.html[RowSelectorClicked] -- 行セレクターがクリックされた後でイベントが発生します。

このイベントを使用して、どの行セレクターがクリックされたのかを判断することができます。
