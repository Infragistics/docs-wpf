= xamDataGrid

== このグループのトピックについて

=== 概要

このセクションには、WPF の  _xamDataGrid_™ コントロールについてのトピックが含まれています。

_xamDataGrid_   は、データを行および列で表として Infragistics® データ プレゼンター ファミリー コントロールです。  _xamGrid_™ コントロールと同様の機能があります。 (このコントロールの詳細な比較については、 link:differences-between-xamgrid-and-xamdatagrid.html[ _xamGrid_   と  _xamDataGrid_   の重要な相違点]を参照してください。)
