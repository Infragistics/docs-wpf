== 選択した単一データ項目の構成

=== 概要

_xamMultiColumnComboEditor_   link:{ApiPlatform}controls.editors.xamcomboeditor{ApiVersion}~infragistics.controls.editors.comboeditorbase`2~selecteditem.html[SelectedItem] プロパティを使用して、選択したデータ項目を構成します。この項目は、使用するデータ モデル内の項目への参照です。

_xamMultiColumnComboEditor_   で項目を選択すると、 link:{ApiPlatform}controls.editors.xamcomboeditor{ApiVersion}~infragistics.controls.editors.comboeditorbase`2~selectionchanged_ev.html[SelectionChanged] イベントが発生します。このイベントには、セクションの link:{ApiPlatform}controls.editors.xamcomboeditor{ApiVersion}~infragistics.controls.editors.selectionchangedeventargs~addeditems.html[AddedItems]/ link:{ApiPlatform}controls.editors.xamcomboeditor{ApiVersion}~infragistics.controls.editors.selectionchangedeventargs~removeditems.html[RemovedItems] を公開するイベント引数 link:{ApiPlatform}controls.editors.xamcomboeditor{ApiVersion}~infragistics.controls.editors.selectionchangedeventargs_members.html[SelectionChangedEventArgs] があります。

_xamMultiColumnComboEditor_   link:{ApiPlatform}controls.editors.xamcomboeditor{ApiVersion}~infragistics.controls.editors.comboeditorbase`2~allowmultipleselection.html[AllowMultipleSelection] プロパティが true に設定されている場合:

• `SelectedItem` が最後に選択された項目を返す

• 新しいデータ オブジェクトに設定した `SelectedItem` により、既存の選択された項目コレクションがクリアされる

注:

[NOTE]
====
`AllowMultipleSelection` プロパティが `false` に設定され、1 つ以上の項目がランタイムで `SelectedItem` プロパティに設定されている場合は、例外が発生します。
====

=== プロパティ設定

以下の表は、任意の構成とそれを管理するプロパティ設定のマップを示します。

[options="header", cols="a,a,a"]

private void MultiColumnComboEditor_OnSelectionChanged(object sender, SelectionChangedEventArgs e)
    {
        if (MultiColumnComboEditor.SelectedItem != null)
        {
            var selectedItem = MultiColumnComboEditor.SelectedItem as Product;
            System.Diagnostics.Debug.WriteLine(String.Format("Product info: {0} {1}", selectedItem.ProductID, selectedItem.ProductName));
        }
}
----

      End If
End Sub

== データ モデルのブール値プロパティの値で選択したデータ項目コレクションの変更

=== 概要

_xamMultiColumnComboEditor_   で項目の現在の状態がブール値で決定される基本データ モデルのプロパティを使用して、選択済み/未選択のデータ項目を構成します。基本データ モデルのプロパティ名が link:{ApiPlatform}controls.editors.xamcomboeditor{ApiVersion}~infragistics.controls.editors.comboeditorbase`2~isselectedmemberpath.html[IsSelectedMemberPath] プロパティに設定されます。

`IsSelectedMemberPath` がデータ モデルのプロパティ パスに設定されると、指定プロパティの値が`true` に設定されたデータ項目で link:{ApiPlatform}controls.editors.xamcomboeditor{ApiVersion}~infragistics.controls.editors.comboeditorbase`2~selecteditems.html[SelectedItems] コレクションが生成されます。基本データ モデルが `INotifyPropertyChanged` インターフェイスをサポートする場合、指定プロパティの値の変更が  _xamMultiColumnComboEditor_   項目の選択状態にも反映されます。

注:

[NOTE]
====
_xamMultiColumnComboEditor_   のデータ項目の選択済み/未選択を決定するデータ モデルのプロパティは、`bool` 型であることが必要です。
====

[NOTE]
====
複数選択を有効にし、`IsSelectedMemberPath` を指定する場合は、最初に `AllowMultipleSelection` プロパティを XAML で定義する必要があります。
====

[NOTE]
====
バインドされたデータ ソースのサイズによっては、`IsSelectedMemberPath` プロパティの使用が  _xamMultiColumnComboEditor_   コントロールのパフォーマンスに影響を与えます。
====

== 選択した単一の値の構成

=== 概要

link:{ApiPlatform}controls.editors.xamcomboeditor{ApiVersion}~infragistics.controls.editors.comboeditorbase`2~selectedvaluepath.html[SelectedValuePath] と共に  _xamMultiColumnComboEditor_   link:{ApiPlatform}controls.editors.xamcomboeditor{ApiVersion}~infragistics.controls.editors.comboeditorbase`2~selectedvalue.html[SelectedValue] プロパティを使用して選択された項目の値を構成し、選択に使用される link:{ApiPlatform}controls.editors.xamcomboeditor{ApiVersion}~infragistics.controls.editors.comboeditorbase`2~selecteditem.html[SelectedItem] メンバーへのパスを指定します。

データ項目コレクションのいくつかの値が、指定された `SelectedValue` と同じ値である場合は、最初の項目が選択されます。

注:

[NOTE]
====
`SelectedValuePath` プロパティは、`SelectedValue` プロパティより前に設定する必要があります。
====

=== プロパティ設定

[options="header", cols="a,a,a"]

== 選択した値のコレクションの構成

=== 概要

link:{ApiPlatform}controls.editors.xamcomboeditor{ApiVersion}~infragistics.controls.editors.comboeditorbase`2~selectedvalues.html[SelectedValues] プロパティを使用して、選択した項目の値のコレクションを構成します。複数の選択がコントロールで有効な場合に非常に便利です。

選択に使用するデータ項目プロパティのパスは、`SelectedValuePath` によって指定されます。

注:

[NOTE]
====
`SelectedValuePath` で指定されたデータ項目プロパティには、一意の値が入ります。データ項目コレクションのいくつかの値が、指定された `SelectedValues` のいずれかと同じ値である場合は、最初の項目が選択されます。
====
