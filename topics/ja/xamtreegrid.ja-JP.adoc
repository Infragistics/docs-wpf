= xamTreeGrid

== このグループのトピックについて

=== 概要

このセクションには、WPF の  _xamTreeGrid_™ コントロールについてのトピックが含まれています。

link:{ApiPlatform}datapresenter{ApiVersion}~infragistics.windows.datapresenter.xamtreegrid.html[xamTreeGrid] は、Data Presenter Family のコントロールで、データをツリー グリッド レイアウトに配置します。このコントロールは基本的に、動的な切り替えができない単一のビュー (link:{ApiPlatform}datapresenter{ApiVersion}~infragistics.windows.datapresenter.treeview.html[TreeView]) を実装する link:{ApiPlatform}datapresenter{ApiVersion}~infragistics.windows.datapresenter.xamdatapresenter.html[xamDataPresenter] です。

以下のスクリーンショットは、"Name" 列にあるツリーを持つ  _xamTreeGrid_   コントロールを示します。
