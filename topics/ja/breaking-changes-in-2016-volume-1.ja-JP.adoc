== TriangulationSource

[[_Ref421542581]]

=== TriangulationSource クラスが共有アセンブリに移動されました。

link:{ApiPlatform}datavisualization{ApiVersion}~infragistics.controls.charts.triangulationsource.html[TriangulationSource] クラスの場所が xamGeographicMap アセンブリから削除され、共有 DataVisualization アセンブリに移動されました。

[options="header", cols="a,a"]
|====
|古いアセンブリ|新しいアセンブリ

|InfragisticsWPF4.Controls.Maps.XamGeographicMap
|InfragisticsWPF4.DataVisualization

|====

[NOTE]
====
DataVisualization アセンブリが既に xamGeographicMap に必要であったため、TriangulationSource クラスの名前空間は変更されませんでした。実装前に必要なその他の作業はありません。
====

== xamDataPresenter

[[_Ref421542577]]

=== デフォルトのプロパティ値の変更

link:{ApiPlatform}datapresenter{ApiVersion}~infragistics.windows.datapresenter.fieldlayoutsettings~allowfieldmoving.html[AllowFieldMoving] プロパティのデフォルト値は、`Yes` から `WithinLogicalRow` に変更されました。

[options="header", cols="a,a,a"]
|====
|プロパティ|以前の表示|新しい表示

|`AllowFieldMoving`
|`はい`
|`WithinLogicalRow`

|====

*回避方法*

以前の動作を保持する場合は、`AllowFieldMoving` プロパティを `Yes` に設定します。

==== 関連トピック:

link:{ApiPlatform}datapresenter{ApiVersion}~infragistics.windows.datapresenter.fieldlayoutsettings~allowfieldmoving.html[AllowFieldMoving]

=== Fields コレクションの項目にアクセスできない

特定のフィールド レイアウトの初期化前に `FieldLayout` の `Fields` コレクションに XAML で定義されるフィールドにアクセスできない場合があります。たとえば、これは xamDataGrid の `Loaded` イベントにアクセスできない場合などです。

*回避策:*

. `FieldLayout` が初期化される前に、フィールド定義を含む `FieldLayout` の `FieldItems` コレクションを使用できます。`FieldItems` コレクションが `FieldItem` 型を使用するため、`Field` 固有のプロパティにアクセスするには、`Field` 型にキャストする必要が可能です。

. xamDataGrid の `FieldLayoutInitialized` イベントを使用すると、特定の `FieldLayout` が初期化されたときに通知します。つまり、そのときに `Fields` および `FieldItems` コレクションが生成され、同期化されます。

== xamSchedule

[[_Ref421542579]]

=== 繰り返しラベルのデフォルト表示の変更

xamSchedule の定期的な予定ダイアログの説明ラベルがドイツ語およびフランス語でもデフォルトで提供されます。

[options="header", cols="a,a,a"]
|====
|言語|以前の表示|新しい表示

|英語
|Visible
|Visible

|ドイツ語
|Hidden
|Visible

|フランス語
|Hidden
|Visible

|スペイン語
|Hidden
|Hidden

|日本語
|Hidden
|Hidden

|====

*回避方法*

以前の動作を保持する場合は、表示を構成する文字列リソースをリソース ファイル (.resx) に追加し、このリソースファイルを xamSchedule の任意のビューに以下の関連トピックに記載された手順に従って登録します。

==== 関連トピック:

link:xamschedule-using-activitydialogs.html[アクティビティ ダイアログ]
