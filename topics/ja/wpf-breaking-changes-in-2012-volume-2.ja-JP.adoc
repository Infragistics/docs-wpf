= 2012 Volume 2 の重大な変更点

== WPF で廃止されたコントロール

=== 廃止されたコントロール

2012 Volume 2 リリースでは、以下のコントロールは {ProductName} 製品で提供されていません。

* xamChart™
* xamTileView™

== WPF の XAML 名前空間マッピング

=== 概要

2012 volume 2 のリリースでは、WPF アセンブリの更新をマッピング XAML 名前空間で統一性を提供します。各アセンブリによって定義されるため、特に (静的) `Resources` クラスが変更されています。新しく作成し、共有する名前空間は link:{RootAssembly}{ApiVersion}~infragistics.shared.dynamicresourcestring_members.html[DynamicResourceString] プロパティのコピーを含みます。

以下は、廃止された URL です。

* http://infragistics.com/DataPresenter
* http://infragistics.com/DockManager
* http://infragistics.com/Editors
* http://infragistics.com/OutlookBar
* http://infragistics.com/Reporting
* http://infragistics.com/Themes
* http://infragistics.com/Windows

WPF 名前空間へのマッピングの新しい URL:

* `http://schemas.infragistics.com/xaml/wpf`

以下のテーブルは、以前のクラス名を本リリースの新しいクラス名にマッピングするリストです。
