== リッチ テキスト形式のコンテンツのインポート

=== 概要

link:{ApiPlatform}documents.richtextdocument{ApiVersion}~infragistics.documents.richtext.richtextdocument_members.html[RichTextDocument] の `LoadFromRtf` 拡張メソッドを使用して、 _xamRichTextEditor_   コントロール内のリッチ テキスト形式のコンテンツをストリームからインポートします。

=== プロパティ設定

以下の表では、各プロパティ設定の構成です。

[options="header", cols="a,a,a"]
|====
|目的:|このメソッドを使用|次の引数を提供:

|リッチ テキスト形式のコンテンツのインポート
|`LoadFromRtf`
|
* リッチ テキスト形式のコンテンツを含むストリーム オブジェクト 

* link:{ApiPlatform}documents.richtextdocument.rtf{ApiVersion}~infragistics.documents.richtext.rtf.rtfloadoptions_members.html[RtfLoadOptions] タイプのオプション オブジェクトを読み込むコンテンツ。必須ではない。 

|====

[[_Ref362603145]]
== リッチ テキスト形式のコンテンツのエクスポート

=== 概要

`RichTextDocument` の `SaveToRtf` 拡張メソッドを使用して、 _xamRichTextEditor_   のリッチ テキスト形式のコンテンツをストリームにエクスポートします。

=== プロパティ設定

以下の表では、各プロパティ設定の構成です。

[options="header", cols="a,a,a"]
|====
|目的:|このメソッドを使用|次の引数を提供:

|リッチ テキスト形式のコンテンツのエクスポート
|`SaveToRtf`
|
* リッチ テキスト形式のコンテンツをエクスポートするストリーム オブジェクト 

* link:{ApiPlatform}documents.richtextdocument.rtf{ApiVersion}~infragistics.documents.richtext.rtf.rtfsaveoptions_members.html[RtfSaveOptions] タイプのオプション オブジェクトを保存するコンテンツ。必須ではない。
