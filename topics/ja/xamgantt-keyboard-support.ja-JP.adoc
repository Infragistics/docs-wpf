== _xamGantt_   のキーボード サポート

[[_Ref333238933]]

=== 概要

_xamGantt_   コントロールはいくつかの事前定義のキーボード ショートカットを提供し、特定のユーザー アクションを素早く実行できるようになります。

以下のショートカットは、Microsoft Project 2010 のキーボード ショートカットに相当するものです。

注:

[NOTE]
====
これらのショートカットは、Silverlight ホストのアプリケーションでは動作しません。
====

[[_Ref333238937]]

=== xamGantt キーボード サポートの概要

以下の表は、ユーザー アクションと、それに対応するキーボード ショートカットをまとめたものです。

[options="header", cols="a,a,a"]
|====
|目的|方法|詳細

|選択したタスクのインデント
|Alt + Shift + Right
|選択したタスクのインデントを行います。

|選択したタスクをアウトデント
|Alt + Shift + Left
|選択したタスクのアウトデントを行います。

|選択したタスク モードを手動スケジュールに切り替え
|Ctrl + Shift + M
|選択したタスク モードを手動に変更します。

|選択したタスク モードを自動スケジュールに切り替え
|Ctrl + Shift + A
|選択したタスク モードを自動に変更します。

|並べ替えした列を初期順序にリセット
|Shift + F3
|並べ替えした列を初期順序にリセットします。

|選択したサマリ タスクの展開
|Alt + Shift + Plus
|選択したサマリ タスクが展開します。

|選択したサマリ タスクを折り畳む
|Alt + Shift + Minus
|選択したサマリ タスクを折り畳みます。

|選択したチャートをスクロールしてプロジェクト開始日を表示
|Alt + Home
|タイムスケールがスクロールし、プロジェクト開始日が表示されます。

|選択したチャートをスクロールしてプロジェクト終了日を表示
|Alt + End
|タイムスケールがスクロールし、プロジェクト終了日が表示されます。

|選択したチャートをスクロールしてタスク開始日を表示
|Ctrl + Shift + F5
|タイムスケールがスクロールし、選択したタスクの予定開始日が表示されます。

|新しいタイムスケール項目にスクロール
|Alt+ Right
|タイムスケールが、(最小のタイムスケール間隔に基づいて) 小さい間隔で右へスクロールします。

|前のタイムスケール項目にスクロール
|Alt+ Left
|タイムスケールが、(最小のタイムスケール間隔に基づいて) 小さい間隔で左へスクロールします。

|次のタイムスケール ページにスクロール
|Alt + PageUp
|タイムスケールが、(表示可能な領域に基づいて) 大きい間隔で右へスクロールします。

|前のタイムスケール ページにスクロール
|Alt + PageDown
|タイムスケールが、(表示可能な領域に基づいて) 大きい間隔で左へスクロールします。

|_xamGantt_ グリッド内の複数選択
|SHIFT + マウス クリック
|連続した複数項目を選択します。

|_xamGantt_ グリッド内の複数選択
|CTRL + マウス クリック
|不連続の複数項目を選択します。

|====
