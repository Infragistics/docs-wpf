以下のコードは、ドキュメント ビューの分割状態を取得する方法を示しています。

*C# の場合:*

[source,csharp]

Dim split As Boolean = Me.xamRichTextEditor1.IsDocumentViewSplit

== 現在の分割状態を設定

=== 概要

ドキュメント ビューの分割状態を設定できます。ドキュメントを分割した場合、 _xamRichTextEditor_   に高さの初期値が等しい 2 つのドキュメント ビューが作成されます。(高さはプログラムで制御できませんが、ユーザー側での調整は可能です。)ドキュメント ビューの分割を無効にすると、下側のビューが残ります。

=== プロパティ設定

[options="header", cols="a,a,a"]
|====
|目的:|使用するプロパティ:|操作:

|ドキュメント ビューの分割状態を設定
|`IsDocumentViewSplit`
|true または false に設定

|====

=== 例

以下のコードは、ドキュメント ビューの分割状態を設定する方法を示しています。

*C# の場合:*

[source,csharp]
----
this.xamRichTextEditor1.IsDocumentViewSplit = true;
----

*Visual Basic の場合:*

[source,vb]
----
Me.xamRichTextEditor1.IsDocumentViewSplit = True
----
