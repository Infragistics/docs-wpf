== 概要

=== タスク クラスの概要

link:{ApiPlatform}controls.schedules.xamgantt{ApiVersion}~infragistics.controls.schedules.project_members.html[Project] クラスは  _xamGantt_   コントロールの主要クラスです。

以下を含むプロジェクトのデータを管理します:

* タスク
* リソース
* カレンダー

Microsoft’s Project 2010 エンジンに似た、定義済みのスケジューリング エンジンがあり、制約、期限、先行処理、後続処理に基づいてタスクの日付を調整します。
