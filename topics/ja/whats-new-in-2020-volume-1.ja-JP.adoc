= 2020 Volume 1 の新機能

=== .NET Core 3.1 サポート

Infragistics コントロールの使用で新しい環境がサポートされます。バージョン 2020.1 以降、アセンブリで NuGet を介した .NET Core 3.1 環境がサポートされます。

.NET Core 3.1 については、link:https://docs.microsoft.com/ja-jp/dotnet/core/whats-new/dotnet-core-3-1[Microsoft サイト] をご確認ください。 

Infragistics NuGet パッケージを使用するための環境設定については、link:nuget-feeds.html[NuGet フィード] および link:nuget-packages.html[NuGet パッケージ]のトピックを参照してください。
