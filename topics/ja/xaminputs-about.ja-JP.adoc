= xamInputs について

== トピックの概要

=== 目的

このトピックは、xamInputs グループのコントロールを紹介します。

* コントロールの概要
* xamCurrencyInput
* xamDateTimeInput
* xamNumericInput
* xamMaskedInput

== コントロールの概要

xamInputs グループのコントロールは、データ入力中にユーザーに指示を与えるために使用され、また入力されたデータがあらかじめ定義された基準のいくつかを満たすかどうかを確認する機会を提供します。入力コントロールは、スタンドアロンのテキスト ボックスとして使用したり、他のコントロールと組み合わせて使用することができます (たとえば、xamGrid コントロールのセルとして)。

以下に、xamInputs グループのコントロールを簡単に説明します。

== xamCurrencyInput

xamCurrencyInput は、入力のパターンを提供することによって、エンドユーザーによる通貨データの入力と編集を容易にする、テキスト ボックス コントロールです。また、マスクが指定されていない場合、コントロールはシステムの通貨形式設定を使用して、そこに含まれるデータを表示します。

image::images/xamInputs_xamCurrencyInput_About.png[]

== xamDateTimeInput

xamDateTimeInput は、ユーザーによる DateTime データの入力や編集を支援する、テキスト ボックス コントロールです。コントロールはマスク プロパティを公開し、それによって入力または編集された値の形式を指定できます。また、xamDateTimeInput はカレンダードロップダウンを提供します。これによってユーザーは、テキストボックスにデータ値を明示的に入力する必要なく、直接日付を選択することができます。

image::images/xamInputs_xamDateTimeInput_About.png[]

* link:xamdatetimeinput-about.html[xamDateTimeInput について]

== xamNumericInput

xamNumericInput は、ユーザーの数値入力と編集を支援するためにマスクを使用する、テキスト ボックス コントロールです。コントロールのプロパティを使って、そのコントロールの検証ルールを定義することができます。

image::images/xamInputs_xamNumericInput_About.png[]

==== 関連トピック

link:xamnumericinput-about.html[xamNumericInput について]

link:xamnumericinput-using.html[xamNumericInput の使用]

== xamMaskedInput

xamMaskedInput は、他の xamInputs が生成されるベース コントロールです。そのため、これには最も広範なアプリケーションが含まれています。マスク、値のタイプ、およびその他のプロパティを構成することができ、それによってデータに対する制限を設定できます。

image::images/xamInputs_xamMaskedInput_About.png[]
