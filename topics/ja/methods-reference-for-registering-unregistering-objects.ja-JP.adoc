== オブジェクトの登録/登録解除のためのメソッド参照

[[_Ref320888387]]

=== 概要

Undo/Redo Framework™ は、オブジェクトを UndoManager インスタンスに登録/登録解除するためのメソッドを提供します。

[[_Ref320888392]]

=== メソッド参照の概要

以下の表で、オブジェクトの登録/登録解除のためのメソッドの目的と機能を簡単に説明します。

[options="header", cols="a,a"]
|====
|メソッド|説明

| link:{ApiPlatform}undo{ApiVersion}~infragistics.undo.undomanager~fromreference.html[FromReference]
|この静的なメソッドは、関連付けられた link:{ApiPlatform}undo{ApiVersion}~infragistics.undo.undomanager_members.html[UndoManager] インスタンスへの参照を取得します。このメソッドはすでに登録されたオブジェクトをパラメーターとして取得します。

| link:{ApiPlatform}undo{ApiVersion}~infragistics.undo.undomanager~registerreference.html[RegisterReference]
|オブジェクトを特定の `UndoManager` インスタンスに関連付けます。 

このメソッドは以下の場合に役に立ちます: 

* `UndoManager` インスタンスへのオブジェクト強参照を保持したくない場合 

* link:{ApiPlatform}undo{ApiVersion}~infragistics.undo.undomanager~current.html[Current] プロパティによって返される共有のスレッド静的な UndoManager インスタンスを使用したくない場合 

これらの参照は弱参照を介して保持されるため、これらの登録解除は必須ではありません。

| link:{ApiPlatform}undo{ApiVersion}~infragistics.undo.undomanager~unregisterreference.html[UnregisterReference]
|オブジェクトの `UndoManager` インスタンスへの登録を削除します。

|====
