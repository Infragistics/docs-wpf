= xamColorPicker について

xamColorPicker™ コントロールは、定義済みの色パレットからエンド ユーザーが色を選択できるエディターです。コントロールはドロップタウン コンテナーにパレットを公開し、定義済みのパレットを含みます。また、パレットのカスタム セットを定義することもできます。xamColorPicker コントロールによって返された値は、選択された色です。

*主要機能*

カスタム パレット - xamColorPicker コントロールによって、アプリケーションに適したパレットが提供されていない場合に独自パレットを定義できます。

完全にカスタマイズ可能 - xamColorPicker コントロールのレイアウトと表示は完全にカスタマイズ可能です。コントロールをスタイル可能で、パレットのテキストを変更したり、パレットの表示/非表示を変更することさえ可能です。

カラー モデル - xamColorPicker コントロールは RGB、CMYK および HLS の各カラー モデルをサポートします。
