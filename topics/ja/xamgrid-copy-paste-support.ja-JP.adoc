= コピー/貼り付けのサポート

xamGrid コントロールは、コピー/貼り付け機能をサポートします。

この機能によって、xamGrid コントロールから選択したデータをクリップボードにコピーし、その後、適切な形式でそれを Microsoft® Excel® ワークシートに貼り付けることができます。Microsoft® Excel® ワークシートからデータをクリップボードにコピーし、その後、xamGrid コントロールにも貼り付けることができます。

コピー操作は "CTRL + C" キーボード ショートカットに割り当てられ、貼り付け操作は "CTRL + V" の組み合わせに割り当てられます。
