= Data Presenter Family コマンドの使用



xamDataCarousel、xamDataGrid、および xamDataPresenter コントロールは、コントロールで特定のアクションを実行するために使用することができる固有のコマンド バインディング セットを提供します。バインディングに使用可能なこれらすべてのコントロールは、 link:{ApiPlatform}datapresenter{ApiVersion}~infragistics.windows.datapresenter.datapresentercommands_members.html[DataPresenterCommands] クラスに存在します。

使用可能なコマンドのいくつかは以下のとおりです。

* CellAbove
* CellRight
* DeleteSelectedDataRecords
* ExpandRecord
* RecordAbove
* StartEditMode
* その他多数...

これらのコマンドが動作するのを確認するには、xamDataGrid カテゴリーの下の xamFeatureBrowser にある、「xamDataGrid Commands」というサンプルを表示します。
