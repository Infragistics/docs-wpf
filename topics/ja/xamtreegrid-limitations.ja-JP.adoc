== 制限

=== 制限の概要表

以下の表に、 _xamTreeGrid_   コントロールの制限の概要を示します。概要表の後にすべての制限の詳細な説明があります。

==== 凡例:

image:images/Yes.png[] - 回避策

image:images/No.png[] - 既知の回避策なし

image:images/Fix.png[] - 修正予定です

[options="header", cols="a,a,a"]
|====
|制限|説明|状態

|レコードの追加はサポートしていません
|_xamTreeGrid_ コントロールは、レコードの追加に使用する UI を提供していないため、 link:{ApiPlatform}datapresenter{ApiVersion}~infragistics.windows.datapresenter.fieldlayoutsettings.html[FieldLayoutSettings] の link:{ApiPlatform}datapresenter{ApiVersion}~infragistics.windows.datapresenter.fieldlayoutsettings~allowaddnew.html[AllowAddNew] プロパティは無視されます。
|image::images/No.png[]

|展開可能なフィールド レコードはサポートしていません
|_xamTreeGrid_ コントロールは、 _xamDataGrid_ と同様、展開可能なフィールドレコードをサポートしていません。代わりに、単一フィールド レイアウトを使用して描画される展開可能なレコードをサポートしています。そのため、 link:{ApiPlatform}datapresenter{ApiVersion}~infragistics.windows.datapresenter.datapresenterbase~fieldsettings.html[FieldSettings] オブジェクトの "ExpandableFieldRecord…" で始まるすべてのプロパティは、このコントロールで機能しません。
|image::images/No.png[]

|グループはサポートしていません
|_xamTreeGrid_ コントロールは、レコードのグループをサポートしていないため、"GroupBy…" プロパティはすべて無視されます。
|image::images/No.png[]

|セル内のラベル位置はサポートしていません
|FieldLayoutSettings の link:{ApiPlatform}datapresenter{ApiVersion}~infragistics.windows.datapresenter.fieldlayoutsettings~labellocation.html[LabelLocation] プロパティに対する link:{ApiPlatform}datapresenter{ApiVersion}~infragistics.windows.datapresenter.labellocation.html[InCells] の設定は、`SeperateHeader` として解決されます。
|image::images/No.png[]

|UseCollectionView フィルタリングはサポートしていません。
|_xamTreeGrid_ コントロールは、 link:{ApiPlatform}datapresenter{ApiVersion}~infragistics.windows.datapresenter.fieldlayoutsettings~filterevaluationmode.html[FilterEvaluationMode] プロパティの link:{ApiPlatform}datapresenter{ApiVersion}~infragistics.windows.datapresenter.filterevaluationmode.html[UseCollectionView] フィルタリング オプションをサポートしていません。`UseCollectionView` オプションを設定した場合、`Auto` として解決されます。
|image::images/No.png[]

|====
