= カルーセル ナビゲーション 要素の位置を変更



xamCarouselPanel、xamCarouselListBox™、xamDataCarousel™、および xamDataPresenter™ のカルーセル ビュー コントロールはすべて、コントロールの右下隅に表示するナビゲーション要素を持っています。ナビゲーション要素の位置を変更したい場合があります。変更するには、 link:{RootAssembly}{ApiVersion}~infragistics.windows.controls.carouselpanelnavigator.html[CarouselPanelNavigator] をターゲットとするスタイルを作成し、Horizontal および Vertical の配置の setter プロパティを指定します。

以下の XAML は、CarouselPanelNavigator オブジェクトの HorizontalAlignment プロパティと VerticalAlignment プロパティを設定します。この XAML は、ページ/ウィンドウ リソース セクションに配置してください。

*XAML の場合:*

----
<Style TargetType="{x:Type igWindows:CarouselPanelNavigator}">
        <Setter Property="HorizontalAlignment" Value="Left"/>
        <Setter Property="VerticalAlignment" Value="Top"/>
</Style>
----
