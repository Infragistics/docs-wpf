== はじめに

=== データ バインディングの概要

_xamGantt_   コントロールは、データ バインディングの 2 つのオプションを提供します。

* link:{ApiPlatform}controls.schedules.xamgantt{ApiVersion}~infragistics.controls.schedules.xamgantt_members.html[XamGantt] の link:{ApiPlatform}controls.schedules.xamgantt{ApiVersion}~infragistics.controls.schedules.projectcontrolbase~project.html[Project] プロパティを通して Project インスタンスに設定
* link:{ApiPlatform}controls.schedules.xamgantt{ApiVersion}~infragistics.controls.schedules.listbackedproject_members.html[ListBackedProject] およびその link:{ApiPlatform}controls.schedules.xamgantt{ApiVersion}~infragistics.controls.schedules.listbackedproject~taskitemssource.html[TaskItemsSource] プロパティを通して任意のカスタム タスク コレクションに設定

[[_Ref334083448]]

== データ ソースにバインドする

=== サポートされるデータ ソースの要約

_xamGantt_   コントロールは以下のデータ ソースをサポートしています。

[options="header", cols="a,a"]
|====
|データ ソース|バインディング

|プロジェクト
|_xamGantt_ コントロールを、 link:{ApiPlatform}controls.schedules.xamgantt{ApiVersion}~infragistics.controls.schedules.project_members.html[Project] インスタンスにバインドできます。 

詳細については、 link:xamgantt-binding-to-data-using-project.html[プロジェクトを使用したデータへのバインディング]を参照してください。

|カスタム タスクのフラット コレクション
| link:{ApiPlatform}controls.schedules.xamgantt{ApiVersion}~infragistics.controls.schedules.listbackedproject_members.html[ListBackedProject] クラスを使って、 _xamGantt_ コントロールをカスタム タスクのフラット データ コレクションにバインドできます。 

詳細については、 link:xamgantt-binding-arbitrary-tasks-collection-listbackedproject.html[ListBackedProject を使用した任意のタスク コレクションへのバインディング]を参照してください。

|MS Project 2010 XML ファイル
|Project クラス link:{ApiPlatform}controls.schedules.xamgantt{ApiVersion}~infragistics.controls.schedules.project~loadfromprojectxml.html[LoadFromProjectXml] メソッドを使って、 _xamGantt_ コントロールを MS Project 2010 XML ファイルからのデータにバインドできます。 

詳細については、 link:xamgantt-loading-project-plan-from-ms-project-xml-file.html[MS Project XML ファイルからのプロジェクト計画の読み込み]トピックを参照してください。

|====
