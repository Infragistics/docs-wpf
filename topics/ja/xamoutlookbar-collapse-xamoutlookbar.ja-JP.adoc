= xamOutlookBar の縮小



Microsoft® Outlook® のナビゲーション ペインと全く同じように、エンドユーザーは画面領域を最大限にするために実行時に xamOutlookBar™ を縮小することができます。エンドユーザーが xamOutlookBar を縮小することに加えて、 link:{ApiPlatform}outlookbar{ApiVersion}~infragistics.windows.outlookbar.xamoutlookbar~isminimized.html[IsMinimized] プロパティを True に設定することで xamOutlookBar をプログラム的に縮小することも可能です。

以下のコード例は、xamOutlookBar の縮小方法を示します。

image::images/xamOutlookBar_Collapse_xamOutlookBar_01.png[]

*XAML の場合:*

----
<igOutlookBar:XamOutlookBar Name="xamOutlookBar1"
               HorizontalAlignment="Left"
               IsMinimized="True" />
----

*Visual Basic の場合:*

----
Me.xamOutlookBar1.IsMinimized = True
----

*C# の場合:*

----
this.xamOutlookBar1.IsMinimized = true;
----
