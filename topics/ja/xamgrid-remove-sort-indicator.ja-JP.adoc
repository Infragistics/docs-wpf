= 並べ替えインジケーターの削除

xamGrid コントロールの並べ替えインジケーターは、特定の列の並べ替えの方向を示します。ほとんどの状況では、並べ替えインジケーターが上を指している場合は列が昇順で並べ替えることを示し、並べ替えインジケーターが下を指している場合は列が降順で並べ替えることを示します。ただし、並べ替えインジケーターをグリッドに表示させたくない状況もあります。これは xamGrid コントロールで簡単に設定できます。

以下のコードは、並べ替えインジケーターを並べ替えた列から削除する方法を示しています。

*XAML の場合:*

----
<ig:XamGrid.SortingSettings>
   <ig:SortingSettings ShowSortIndicator="/>
</ig:XamGrid.SortingSettings>
----

*Visual Basic の場合:*

----
Me.MyGrid.SortingSettings.ShowSortIndicator = False
----

*C# の場合:*

----
this.MyGrid.SortingSettings.ShowSortIndicator = false;
----
