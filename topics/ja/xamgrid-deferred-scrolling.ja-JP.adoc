= 遅延スクロール

遅延スクロールは xamGrid コントロールの UI 機能で、これによってエンド ユーザーは高速かつ効率的に大量のデータを検索できます。

遅延スクロールが有効である場合、データをスクロールすると、現在のつまみの位置に対応する行についての情報を含むカスタマイズ可能なプレビューがエンドユーザーに表示されます。エンドユーザーがスクロールを停止すると、その位置のデータが表示されます。xamGrid のスクロールバーは、データセット全体を表します。
