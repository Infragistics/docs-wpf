= 署名済みのアセンブリと未署名のアセンブリ

{ProductName} フル製品のインストーラは、未署名のアセンブリを bin フォルダーとグローバル アセンブリ キャッシュ (GAC) にインストールします。さらに、署名済みのアセンブリを含む zip されたファイルも以下のフォルダーで使用できます: {InstallPath}\Bin - Signed。

以下は、署名済みのアセンブリの使用のシナリオと制限のリストです。

* XAML Browser Application (XBAP) を配備している場合には、署名済みのアセンブリを使用する必要があります。
* 署名済みのアセンブリを使用しており、インターネットに接続できない環境に配備する場合、現在の Windows セッション内で初めてアプリケーションを実行する時には、開始に時間がかかります (数秒から、ロードされる署名済みのアセンブリの数によっては 2 分以上かかる場合があります)。オペレーティング システムが各署名済みのアセンブリをロードするとき、署名済みのアセンブリに添付されている証明書が無効でないことを検証しようとします。インターネット接続がないときには、この作業を依然として試みるため試行がタイムアウトするまで戻りません。
