= Generic.xaml ファイルの場所

Generic.xaml ファイルには {ProductName} コントロールを描画するすべてのスタイル、テンプレート、およびブラシが含まれます。これらのスタイルは、カスタムスタイルを作成するための土台として使用したり、特定のスタイルをアプリケーションで変更したりすることができます。

デフォルトのテーマ ファイルは、link:https://github.com/Infragistics/wpf-resources/tree/main/DefaultStyles[WPF Resources  GitHub リポジトリ]で公開されています。
