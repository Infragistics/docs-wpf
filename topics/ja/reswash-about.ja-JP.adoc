= リソース ウォッシャーについて

== ResourceWasher の構成の概要

pick:[sl=" link:{RootAssembly}{ApiVersion}~infragistics.resourcewasher.html[ResourceWasher]"]  pick:[wpf=" link:{RootAssembly}{ApiVersion}~infragistics.windows.themes.resourcewasher.html[ResourceWasher]"]  コンポーネントは XAML のクロス プラットフォーム機能です。これには ResourceDictionary のような視覚的表現はありません。このコンポーネントの目的は、ランタイムに単色または複数色で 1 つの ResourceDictionary をウォッシュするための簡単かつ強力な方法を提供することです。

カラー ウォッシングは、新しい色 ( pick:[sl=" link:{RootAssembly}{ApiVersion}~infragistics.resourcewasher~washcolor.html[WashColor]"]  pick:[wpf=" link:{RootAssembly}{ApiVersion}~infragistics.windows.themes.resourcewasher~washcolor.html[WashColor]"]  プロパティ) およびリソースに適用される色の変換 ( pick:[sl=" link:{RootAssembly}{ApiVersion}~infragistics.resourcewasher~washmode.html[WashMode]"]  pick:[wpf=" link:{RootAssembly}{ApiVersion}~infragistics.windows.themes.resourcewasher~washmode.html[WashMode]"]  プロパティ) を指定することによって、リソースの元の色を変更するプロセスです。以下の表は、ウォッシング前後のピボット グリッドを表しています。「After」ピボット グリッドは、デフォルトの  pick:[sl=" link:{RootAssembly}{ApiVersion}~infragistics.washmode.html[WashMode]"]  pick:[wpf=" link:{RootAssembly}{ApiVersion}~infragistics.windows.themes.washmode.html[WashMode]"]   (SoftLightBlend) で Lime でウォッシュされています。
