== エディターのズーム レベルの設定

=== 概要

0.2～4.0 間の type double を使用して、エディターのズーム レベルを設定します (ここで、1.0 は 100% を表します)。

=== プロパティ設定

以下の表では、任意の構成と特定のプロパティ設定のマップを示します。

[options="header", cols="a,a,a"]
|====
|目的:|使用するプロパティ:|設定値

|エディターのズーム レベルの設定
|`ZoomLevel`
|type double の 0.2～4.0 の間の値

|====

[[_Hlk337817761]]

=== 例

以下のコード スニペットは、ビューのズーム レベルを 200% に調整する方法を示します。

*C# の場合:*

[source,csharp]
----
this.xamRichTextEditor.ZoomLevel = 2.0;
----

*Visual Basic の場合:*

[source,vb]
----
Me.xamRichTextEditor.ZoomLevel = 2.0
----

[[_Ref362446360]]
== エディターのズーム レベルの取得

=== 概要

type double の値でエディターのズーム レベルを取得します。

=== プロパティ設定

以下の表では、任意の構成と特定のプロパティ設定のマップを示します。

[options="header", cols="a,a,a"]
|====
|目的:|使用するプロパティ:|操作:

|エディターのズーム レベルを取得する
|`ZoomLevel`
|値を取得

|====

=== 例

以下のコード スニペットは、ビューのズーム レベルを取得する方法を示します。

*C# の場合:*

[source,csharp]
----
double zoom = this.xamRichTextEditor.ZoomLevel;
----

*Visual Basic の場合:*

[source,vb]
----
Dim zoom As Double = Me.xamRichTextEditor1.ZoomLevel
----

=== 概要

エディターのズーム レベルをパーセント値で取得します。

=== プロパティ設定

以下の表では、任意の構成と特定のプロパティ設定のマップを示します。

[options="header", cols="a,a,a"]
|====
