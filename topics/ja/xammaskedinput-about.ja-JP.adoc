= xamMaskedInput について

xamMaskedInput コントロールは、ユーザーのデータ入力を支援するためのマスクを使用する、シンプルなテキスト ボックス コントロールです。マスクを使って、ユーザーの入力を事前定義されたテキスト形式に従って制限することができます。xamMaskedInput コントロール を link:xammaskedinput-using.html[スタンドアロン エディターとして] pick:[sl,wpf,win-rt="または link:xaminputs-using-xaminput-controls-in-xamgrid-cells-for-data-editing.html[xamGrid™ コントロールのセル内で]"] 使用できます。
