== Field、FieldSettings、FieldLayout および FieldLayoutSettings の MVVM プロパティへのバインド

=== 概要

link:{ApiPlatform}datapresenter{ApiVersion}~infragistics.windows.datapresenter.fieldbinding_members.html[FieldBinding] マークアップ拡張子を使用して、 link:{ApiPlatform}datapresenter{ApiVersion}~infragistics.windows.datapresenter.field_members.html[Field]、 link:{ApiPlatform}datapresenter{ApiVersion}~infragistics.windows.datapresenter.fieldsettings_members.html[FieldSettings]、 link:{ApiPlatform}datapresenter{ApiVersion}~infragistics.windows.datapresenter.fieldlayout_members.html[FieldLayout] および link:{ApiPlatform}datapresenter{ApiVersion}~infragistics.windows.datapresenter.fieldlayoutsettings_members.html[FieldLayoutSettings] の MVVM プロパティへのバインドを容易にします。

注:

[NOTE]
====
`FieldBinding` は `Binding` と同じオプションを表示しますが、以下のプロパティは表示しません。

* `AsyncState`
* `BindingGroupName`
* `BindsDirectlyToSource`
* `Delay`
* `ElementName`
* `IsAsync`
* `RelativeSource`
* `Source`
* `ValidatesOnDataErrors`
* `ValidatesOnExceptions`
* `ValidatesOnNotifyDataErrors`
* `ValidationRules`
* `XPath`

====

注:

[NOTE]
====
`FieldBinding` は、`Field` または `FieldLayout` 以外のオブジェクトで使用された場合、またはタイプ `BindingBase` のプロパティ (`Field` の `AlternateBinding` プロパティなど) にバインドされた場合、`XamlParseException` をスローします。
====

=== 例

MVVM プロパティへのバインドは、以下のいずれかの方法でも実装できます。

*XAML の場合:*

[source,xaml]
----
<igDP:Field  Name="Name" Label="{igDP:FieldBinding MyProperty}" />
----

*XAML の場合:*

[source,xaml]
----
<igDP:Field Name="Name" Label="{Binding Path=Owner.DataPresenter.DataContext. MyProperty, RelativeSource={RelativeSource Self}}"  />
----

*XAML の場合:*

[source,xaml]
----
<igDP:Field Name="Name" Label="{Binding Path=DataContext.MyProperty, RelativeSource={RelativeSource Self}}"  />
----
