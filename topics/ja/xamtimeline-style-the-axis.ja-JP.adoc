= 軸のスタイル

xamWebTimeline コントロールの link:{ApiPlatform}controls.timelines.xamtimeline{ApiVersion}~infragistics.controls.timelines.timelineaxis.html[軸] には、ラベル、主目盛および副目盛、主グリッド線および副グリッド線、ペイン、およびつまみが含まれます。これらの要素のすべてを Axis オブジェクトからスタイルできます。

ラベルは各イベントの Time 値を表示します。主グリッド線と副グリッド線は、値の間隔をマークします。主目盛と副目盛は、Axis の線である、ペインの値を示します。つまみはエンドユーザーがイベントの詳細を表示するために移動できる Axis 上のアイコンです。これらの要素は、ユーザーが設定できる個々のスタイル プロパティを持ちます。または Axis オブジェクトの Style プロパティを使用してそれらすべてをスタイルできます。

image::images/XamTimeline_Style_Axis_01.png[]
