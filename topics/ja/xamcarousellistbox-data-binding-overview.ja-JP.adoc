= データ バインディング概要



xamCarouselListBox™ は、xamDataGrid™、xamDataCarousel™、および xamDataPresenter™ がサポートするのと同じようにデータ バインディングをサポートしますが、機能する方法は、データ コントロールとは異なります。データ コントロールは、データソースを実際に問い合わせ、それに基づいて必要なフィールドとレイアウトを生成します。xamCarouselListBox はデータ ソースに問い合わせを行わずにレイアウトを生成します。従うべき xamCarouselListBox にデータ ソースを提供する必要があります。詳細は、 link:xamcarousellistbox-working-with-the-itemtemplate.html[ItemTemplate での作業]を参照してください。
