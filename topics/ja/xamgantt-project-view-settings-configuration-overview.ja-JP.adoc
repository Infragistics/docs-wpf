== 概要

=== ProjectViewSettings クラスの概要

`ProjectViewSettings` クラスは  _xamGantt_   のビューに影響するこれらの設定を提供します。

これらの構成可能な項目は以下の通りです。

* クリティカル タスクの強調表示
* 非稼働日/時間の強調表示
*  _xamGantt_   のチャート セクションにおけるスラック表示表現

[[_Ref334823701]]
