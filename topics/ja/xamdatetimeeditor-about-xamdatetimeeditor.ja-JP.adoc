= xamDateTimeEditor について

xamDateTimeEditor™ コントロールはシンプルなテキスト ボックス コントロールで、エンド ユーザーが正しいフォーマットで日付/時刻の値を入力する支援をするためにマスクを使用します。このマスクを使用して、事前に決定した有効な日付と時刻のフォーマットにエンドユーザーの入力を制限することができます。スタンドアロン エディターとして、または xamDataGrid™ コントロールの Field で xamDateTimeEditor コントロールを使用します。xamDataGrid のフィールドとして、xamDateTimeEditor は特定のグリッド列の適切なフォーマットを維持すると同時に、エンド ユーザーのデータ入力を支援します。

xamDateTimeEditor の link:{ApiPlatform}editors{ApiVersion}~infragistics.windows.editors.xammaskededitor~mask.html[Mask] プロパティは、日付だけ、時刻だけ、または日付と時刻の両方をエディターによって表示および編集するかどうかを指定するために使用できます。
