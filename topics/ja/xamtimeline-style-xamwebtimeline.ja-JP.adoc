= xamTimeline のスタイル設定

xamTimeline コントロールは、xamZoombar コントロールに加えて、 link:{ApiPlatform}controls.timelines.xamtimeline{ApiVersion}~infragistics.controls.timelines.scene.html[Scene]、 link:{ApiPlatform}controls.timelines.xamtimeline{ApiVersion}~infragistics.controls.timelines.title.html[Title]、 link:{ApiPlatform}controls.timelines.xamtimeline{ApiVersion}~infragistics.controls.timelines.timelinelegend.html[Legend] の 3 つのオブジェクトで主に構成されます。Scene はコントロールの主なコンテンツです。これはコントロールのタイムラインの側面を構成する link:{ApiPlatform}controls.timelines.xamtimeline{ApiVersion}~infragistics.controls.timelines.timelineaxis.html[Axis] および TimeSeries を含んでいます。Legend は xamTimeline によって表示されている時刻シリーズを識別するペインです。Title はコントロールの説明的なテキストを表示するために使用されます。最後に、XamZoomBar コントロールはレイアウトの一番下にあり、ズームしてタイムライン上の情報にスクロールする役割を果たします。

image::images/XamTimeline_Style_XamTimeline_01.png[]
