= 異なるデータ タイプのデフォルトのエディター タイプ



xamDataGrid™、xamDataCarousel™、および xamDataPresenter™ コントロールは特定のデータ タイプのためにデフォルトのエディターを登録しています。以下のテーブルは、デフォルトで編集するために登録されているデータ タイプとエディターを示しています。

[options="header", cols="a,a"]
|====
|データ型|エディター

|string
|xamTextEditor™

|bool
|xamCheckEditor™

|DateTime
|xamDateTimeEditor™

|decimal
|xamCurrencyEditor™

|double
|xamNumericEditor™

|float
|xamNumericEditor

|byte
|xamNumericEditor

|sbyte
|xamNumericEditor

|short
|xamNumericEditor

|ushort
|xamNumericEditor

|int
|xamNumericEditor

|uint
|xamNumericEditor

|long
|xamNumericEditor

|ulong
|xamNumericEditor

|====
