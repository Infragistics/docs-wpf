= xamNetworkNode のスタイリング ポイント

=== 概要

_xamNetworkNode_  コントロールを使用し、コントロール プロパティを介してターゲット タイプのデフォルトのルック アンド フィールをカスタマイズします。

=== プレビュー

以下の画像は、 _xamNetworkNode_  のコントロール プロパティを使用して構成可能な、さまざまなターゲットの種類を示しています。これによって、希望するスタイルを使用して、このコントロールに関連するターゲットの種類をカスタマイズできます。

以下に示された xamOverviewDetailPlusPane コントロールは、レイアウト表示の xamNetworkNode のパンとズームを可能にします。

image::images/xamNetworkNode_StylingPoints_1.png[]

== xamNetworkNode スタイリング プロパティ

_xamNetworkNode_   _™_   コントロールのスタイリング プロパティ。

[options="header", cols="a,a,a"]
|====
|ターゲットの種類|Style プロパティ|説明

|XamNetworkNode
|XamNetworkNode.Style
|xamNetworkNode コントロールをスタイリングします。

|NetworkNodeNodeControl
|NetworkNodeNodeControl.Style
|xamNetworkNode コントロールのノードをスタイリングします。

|XamOverviewPlusDetailPane
|XamOverviewPlusDetailPane.Style
|レイアウト ビューのコントロールのパンとズームを可能にする、xamOverviewPlusDetailPane コントロールをスタイリングします。

|====
