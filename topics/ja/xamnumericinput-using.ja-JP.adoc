== xamNumericInput コントロールをアプリケーションに追加

=== xamNumericInput コントロールをアプリケーションの詳細に追加

[start=1]
. *Infragistics 名前空間を追加します。*

.. *必要な参照を追加します。*

xamNumericInput コントロールをアプリケーションに追加するには、以下の NuGet パッケージへの参照をプロジェクトに追加する必要があります。

*** Infragistics.WPF.Controls.Editors.XamMaskedInput

+
NuGet フィードのセットアップと NuGet パッケージの追加の詳細については、link:nuget-feeds.html[NuGet フィード] ドキュメントを参照してください。

.. *Infragistics 名前空間を追加します。*

*XAML の場合:*

[source,xaml]
----
xmlns:ig="http://schemas.infragistics.com/xaml"
----

*Visual Basic の場合:*

[source,vb]
----
Imports Infragistics.Controls.Editors
----

*C# の場合:*

[source,csharp]
----
using Infragistics.Controls.Editors;
----

[start=2]
. *xamNumericInput コントロールを追加します。*

*XAML の場合:*

[source,xaml]
----
<ig:XamNumericInput/>
----

*Visual Basic の場合:*

[source,vb]
----
Dim input As XamNumericInput = New XamNumericInput()
----

*C# の場合:*

[source,csharp]
----
XamNumericInput NumericInput = new XamNumericInput();
----

== xamNumericInput の詳細を修正

[[SetMask]]
[start=1]
. *Mask の設定*

入力コントロールの設定の最も重要な要素は、Mask プロパティの構成です。によって、自分のニーズに合ったマスクを設定します:

*XAML の場合:*

[source,xaml]
----
<ig:XamNumericInput Mask="[Your Mask]"/>
----

*Visual Basic の場合:*

[source,vb]
----
NumericInput.Mask = "[Your Mask]"
----

*C# の場合:*

[source,csharp]
----
NumericInput.Mask = "[Your Mask]";
----

マスクの作成方法の詳細は、 link:xaminputs-masks.html[マスク]トピックを参照してください。

[[InvalidValueBehavior]]
[start=2]
. *無効な場合は、入力された値を取り消します。*

ユーザーが無効な値を入力したときの xamNumericInput の動作方法には、複数のオプションがあります。InvalidValueBehavior プロパティを RevertValue に設定すると、ユーザーが無効な値を入力して入力コントロールを終了すると、常にその値は元の値に戻ります。

[[ClipMode]]
[start=3]
. *すべての追加記号を含め、ユーザーはコントロールのコンテンツをコピーできます。*

ClipMode プロパティを IncludeBoth に設定すると、すべてのプロンプト文字とマスク リテラルを含む、入力コントロールのコンテンツがクリップボードにコピーされます。

[[NullText]]
[start=4]
. *値が null の場合、特定の文字列を表示します。*

コントロールに値が入力されない場合、または値が null 値にバインドされている場合、表示される文字列を設定することができます。NullText プロパティを希望の文字列に設定することで、これを実行することができます。このように、入力内容や、値が入力されていないことを明示的にユーザーに知らせることができます。

[[PromptChar]]
[start=5]
. *下線以外の様々なプロンプト文字を表示します。*

xamNumericInput で使用されるデフォルトのプロンプト文字は下線ですが、PromptChar プロパティを自分が選択した文字に設定することによって、この動作を変更することができます。

[[SpinButtons]]
[start=6]
. *値を増加/減少するためのボタンを表示します。*

xamNumericInput コントロールの値を増加または減少させるためのボタンを表示したい場合、SpinButtonDisplayMode を Always、Focused、または MouseOver に設定する必要があります。

[[SpinIncrement]]
[start=7]
. *増加値の変更。*

SpinIncrement プロパティを指定しない場合は、SpinButtons がクリックされたときに使用されるデフォルトの増加/減少値は 1 になります。ただし、別の値を使用したい場合は、SpinIncrement として希望の値を設定することによって、値を変えることができます。

[[SetMask]]
[[TrimFractionalZeros]]
[start=8]
. *値の最後にある小数のゼロを保持するように入力を設定します。*

xamNumericInput コントロールの値の小数の場所の最後のゼロを表示したい場合は、TrimFractionalZeros プロパティを False に設定します。

[[IsUndoEnabled]]
[start=9]
. *元に戻す/やり直し操作を有効/無効にします。*

xamInputs は CTRL + Z および CTRL + Y キーの組み合わせによってトリガーされた元に戻す/やり直し操作をデフォルトでサポートします。この機能を無効にするには、IsUndoEnabled プロパティを False に設定します。

注:

[NOTE]
====
`IsUndoEnabled` プロパティを無効にすると、元に戻す/やり直すスタックをクリアします。元に戻す/やり直すを無効にした後に再び有効にする場合、元に戻す/やり直すの履歴がクリアされるためコマンドは実行されません。
====
