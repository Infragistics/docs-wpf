== 構文解析

[[_Ref349221033]]

=== 概要

Syntax Parsing Engine は、字句解析で生成されるトークンの構文解析を実施します。この解析の結果は、ドキュメントの文章校正構造を表す構文ツリーです。構文解析は、 link:{ApiPlatform}documents.textdocument{ApiVersion}~infragistics.documents.parsing.grammar~nonterminalsymbols.html[Grammar.NonTerminalSymbols] コレクションの非終端記号の定義に基づいています。
