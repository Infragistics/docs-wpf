== 列と行を非表示 - xamPivotGrid コントロールの視覚要素

[[_Ref319586985]]

=== 視覚要素の要約

以下のスクリーンショットは、列と行の非表示に関連する xamPivotGrid コントロールの視覚要素を示しています。画像の後に、構成可能な要素の一覧を示しています。

image::images/xamPivotGrid_Hiding_Columns_and_Rows_1.png[]

設定可能なビジュアル要素:
[start=1]
. 非表示列チューザー機能

既定で少なくとも 1 つの列が非表示になっている場合、非表示列チューザー機能が表示されます。クリックすると、すべての非表示列のキャプションを記載したドロップダウンが表示されます。ドロップダウンから列を選択すると、その列が表示されるようになります。 
[start=2]
. 非表示行チューザー機能

列チューザー機能同様、行が非表示になっている場合、非表示行チューザー機能が表示されます。

[start=3]
. [Hide column] ボタン

列の非表示が有効な場合、マウスのカーソルを列ヘッダーに移動すると、このボタンが表示されます。列を非表示にするには、ボタンをクリックします。 

[start=4]
. [Hide row] ボタン

[列を非表示にする] ボタンから推測できるように、行の非表示が有効な場合、マウスのカーソルを行ヘッダーに移動すると、特定の行を非表示にするボタンが表示されます。

.注:
[NOTE]
====
既定では、列と行の非表示は *無効になっています* 。
====

[start=5]
. 非表示軸インジケーター (ヒント)

行または列 (軸) が非表示になっている場合、既定では、非表示の行または列を示すため、隣接する 2 つの軸の間の線は太くなります。

== プロシージャ コードからの列と行の表示設定

[[_Ref319587005]]

=== 概要

UI から列と行の非表示を有効にする場合とは別に、プロシージャ コードの該当するプロパティを設定して、列と行を非表示にすることもできます。この場合、xamPivotGrid の link:{ApiPlatform}controls.grids.xampivotgrid{ApiVersion}~infragistics.controls.grids.xampivotgrid~gridlayout.html[GridLayout] で非表示にする列または行を見つけ、その `IsVisible` プロパティを false に設定する必要があります。

[[_Ref319587008]]

=== プロパティ設定

以下の表では、目的の構成をプロパティ設定にマップしています。

[options="header", cols="a,a,a"]
|====
|目的|使用するプロパティ:|次に設定

|列を非表示にする
| link:{ApiPlatform}controls.grids.xampivotgrid{ApiVersion}~infragistics.controls.grids.pivotdatacolumn~isvisible.html[IsVisible]
|`False`

|行を非表示にする
| link:{ApiPlatform}controls.grids.xampivotgrid{ApiVersion}~infragistics.controls.grids.pivotdatarow~isvisible.html[IsVisible]
|`False`

|====

[[_Ref319587012]]
== コード例: プロシージャ コードからの列と行の表示設定

[[_Ref319587015]]

=== 説明

以下のコード スニペットは、xamPivotGrid の列と行すべてにおいて繰り返し使用され、特定の `HeaderText` 値を持つ列と行を非表示にします。

[[_Ref319587018]]

=== コード

*C# の場合:*

[source,csharp]
----
Code
foreach (PivotDataColumn column in pivotGrid.GridLayout.Columns)
{
    if (column.HeaderText == "CY 2002")
        column.IsVisible = false;
}
foreach (PivotDataRow row in pivotGrid.GridLayout.Rows)
{
    if (row.HeaderText == "Berks")
        row.IsVisible = false;
}
For Each column As PivotDataColumn In pivotGrid.GridLayout.Columns
      If column.HeaderText = "CY 2002" Then
            column.IsVisible = False
      End If
Next
For Each row As PivotDataRow In pivotGrid.GridLayout.Rows
      If row.HeaderText = "Berks" Then
            row.IsVisible = False
      End If
Next
----
