= パスを継続



リストで使用可能な最後のレコードが依然として表示されている間、xamDataPresenter™ でレコード コレクションを右または左にナビゲートできます。link:{RootAssembly}{ApiVersion}~infragistics.windows.controls.carouselviewsettings~islistcontinuous.html[IsListContinuous] プロパティを True に設定することによって、コレクションによって継続的にナビゲートすることも可能です。

リストを継続させるためには、以下のコードを使用します。

*XAML の場合:*

----
<igDP:XamDataPresenter.View>
        <igDP:CarouselView>
                <igDP:CarouselView.ViewSettings>
                        <igWindows:CarouselViewSettings IsListContinuous="True" />
                </igDP:CarouselView.ViewSettings>
        </igDP:CarouselView>
</igDP:XamDataPresenter.View>
----

link:xamdatapresenter-getting-started-with-xamdatapresenter.html[xamDataPresenter をアプリケーションに追加]の XamDataPresenter タグ内のコードを上記のコードで置き換えて、この機能を表示します。
