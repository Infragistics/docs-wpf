このセクションのトピックでは、xamPivotGrid™ のさまざまな機能を使用する方法について幅広い観点から説明します。

|xamPivotGrid™ コントロールには、表示されるデータのレイアウトとフォーマットを指定するためのさまざまな方法があります。

|このトピックは xamPivotGrid コントロール内でデータ セルの編集を有効化し、データ ソースを修正するために編集に関連するイベントを処理する方法を説明します。

| link:xampivotgrid-handlingevents.html[イベントの処理]

|xamPivotGrid™ コントロールには、Field Chooser コントロールが組み込まれています。このコントロールによりユーザーは、行、列、フィルター、およびメジャーに追加する階層を素早く選択できます。このセクションのトピックでは、フィールド選択について詳しく説明します。
