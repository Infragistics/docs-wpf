== ドキュメント ビューの分割

=== 概要

`AllowDocumentViewSplitting` を `true` または `false` に設定すると、分割ハンドルが表示または非表示になります。分割ハンドルの表示または非表示を切り替えると、ドキュメント ビューを分割できる機能を効果的に有効または無効にできます。

=== プロパティ設定

[options="header", cols="a,a,a"]
|====
|目的:|使用するプロパティ:|設定値

|ドキュメント ビューの分割の許可
|`AllowDocumentViewSplitting`
|true

|ドキュメント ビューの分割を許可しない
|`AllowDocumentViewSplitting`
|false

|====

[[_Hlk337817761]]

=== 例

以下のコードは、ドキュメント ビューの分割を許可する方法を示しています。

*C# の場合:*

[source,csharp]
----
this.xamRichTextEditor1.AllowDocumentViewSplitting = true;
----

*Visual Basic の場合:*

[source,vb]
----
Me.xamRichTextEditor1.AllowDocumentViewSplitting = True
----
