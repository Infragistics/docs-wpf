== はじめに

=== 元に戻す / やり直しの概要

元に戻す / やり直し機能は、ドキュメントのコンテンツに対する変更またはコードの変更をすべて記録します。最初のドキュメント作成または新規コンテンツの読み込みの場合、読み込みメソッドの記録はデフォルトでオフになります。以下のいずれかの操作で機能がオンになります。

*  _XamRichTextEditor_   コントロールを使用してドキュメントのコンテンツの編集を開始
* ドキュメントの link:{ApiPlatform}documents.richtextdocument{ApiVersion}~infragistics.documents.richtext.richtextdocument~startundologging.html[StartUndoLogging] メソッドの起動

link:{ApiPlatform}documents.richtextdocument{ApiVersion}~infragistics.documents.richtext.richtextdocument_members.html[RichTextDocument] で link:{ApiPlatform}documents.richtextdocument{ApiVersion}~infragistics.documents.richtext.richtextdocument~stopundologging.html[StopUndoLogging] メソッドを起動すると、元に戻すの履歴がクリアされ、元に戻すログが停止します。

== 元に戻すログの開始

=== 概要

プログラムから `StartUndoLogging` メソッドを起動すると、元に戻すログが開始されます。

[[_Hlk337817761]]

=== 例

*C# の場合:*

[source,csharp]
----
this.xamRichTextEditor1.Document.StartUndoLogging();
----

[[_Ref361240385]]
== 元に戻すログの停止

=== 概要

プログラムから StopUndoLogging メソッドを起動すると、元に戻すログを停止できます。元に戻す履歴もクリアされます。

=== 例

*C# の場合:*

[source,csharp]
----
this.xamRichTextEditor1.Document.StopUndoLogging();
----
