== ツリー フィールドの構成の概要

=== ツリー フィールドの概要

link:{ApiPlatform}datapresenter{ApiVersion}~infragistics.windows.datapresenter.xamtreegrid.html[xamTreeGrid] コントロールにより、ツリーを配置するフィールドを指定できます。値は、フィールド レイアウトに設定したフィールドの名前と一致している必要があります。_xamTreeGrid_   の `FlowDirection` プロパティに応じて、ツリーはデフォルトでフィールドのコンテンツの左側または右側に表示されるように配置されます。

=== ツリー フィールドの構成の概要

以下の表は、xamTreeGrid コントロールの構成可能な要素を簡単に説明し、それを構成するプロパティにマップします。

[options="header", cols="a,a,a"]
|====
|構成可能な項目|詳細|プロパティ

|[[_Hlk356484826]] 

ツリー フィールド
|ツリー フィールドの指定に使用されます。
| link:{ApiPlatform}datapresenter{ApiVersion}~infragistics.windows.datapresenter.treeviewsettings~treefieldname.html[XamTreeGrid.ViewSettings.TreeFieldName]

|====

注:

[NOTE]
====
`TreeFieldName` プロパティが指定されていない場合、最初のフィールドまたはプライマリ フィールドがツリー フィールドとして使用されます。
====

=== コード例

以下のコード例では、ツリー フィールドを設定する方法を示します。

*XAML の場合:*

[source,xaml]
----
<igDP:XamTreeGrid>
  <igDP:XamTreeGrid.FieldLayouts>
    <igDP:XamTreeGrid.FieldLayout>
      <igDP:Field Name="ID" Label="Customer ID" />
      <igDP:Field Name="Name" Label="Customer Name" />
      <igDP:Field Name="Address" Label="Customer Address" />
    </igDP:XamTreeGrid.FieldLayout>
  </igDP:XamTreeGrid.FieldLayouts>
  <igDP:XamTreeGrid.ViewSettings>
    <igDP:TreeViewSettings TreeFieldName="Name" />
  </igDP:XamTreeGrid.ViewSettings>
</igDP:XamTreeGrid>
----
